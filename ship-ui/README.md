## 启动说明

> 请确保已安装Node.js，且版本不得低于v18.16.1

* 第一次启动先下载依赖，在`ship-ui`文件夹下(即与`package.json`同级的路径)，打开终端输入`npm install`，下载依赖。
* 依赖下载完成后，在终端输入`npm run dev`启动前端。

* 参考命令

```shell
cd ship-ui
npm run dev
```

在网页大于小于一定程度时，采用不同的css

```
@media (min-width: 750px) {
    .mixJade {
        width: 38%;
        left: 31%;
    }
}

@media (max-width: 750px) {
    .mixJade {
        width: 90%;
        left: 5%;
    }
}
```

并且这里可以使用AND联合条件

```css
/*响应式*/
@media (min-width: 768px) {
  .base-page {
    width: 50vw;
    left: 25vw;
  }
}

@media (min-width: 576px) and (max-width: 768px) {
  .base-page {
    width: 60vw;
    left: 20vw;
  }
}

@media (max-width: 575px) {
  .base-page {
    width: 80vw;
    left: 10vw;
  }
}
```
